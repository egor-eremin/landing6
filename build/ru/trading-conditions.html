Калькулятор трейдера

Онлайн-калькулятор трейдера - это инструмент, который используется для определения уровня свободных средств для планируемой сделки, а также для просчета размера комиссии. Чтобы использовать калькулятор, необходимо указать исходные параметры.

Как работает калькулятор трейдера

  • Инструмент:
    указывайте выбранный финансовый инструмент из списка
  • Цена:
    укажите предположительную цену открытия и закрытия ордера, чтобы узнать возможный результат
  • Объем сделки:
    укажите точный объем сделки в лотах
  • Своп в пунктах:
    просчитывается автоматически исходя из разницы между ценами Ask и Bid
  • Своп в USD:
    просчитывается автоматически исходя из стоимости переноса позиции в ночное время
  • Маржа:
    указанные данные - это минимальный размер свободного депозита, который необходим для осуществления сделки
  • Прибыль:
    указывается вероятная прибыль на основании введенных параметров
